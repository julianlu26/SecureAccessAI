# Demo Q and A Notes

## Likely Questions

- What does the AI part do?
- How is RBAC enforced?
- How do you detect suspicious login behaviour?
- What would you improve next?

## Suggested Short Answers

- The current AI-related module is a rules-based MVP for risk monitoring and behaviour analysis.
- RBAC is enforced through roles, permissions, and middleware checks on protected routes.
- Suspicious behaviour is detected through failed attempts, rate limiting, and IP-change signals.
- Next improvements would include a richer dashboard, stronger persistence, and more advanced anomaly detection.
